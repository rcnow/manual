Глава 1 - Типы и переменные


Типы

В JavaScript существут несколько типов:

  • Строка (string)
  • Число (number)
  • Логический (boolean)
  • Ничто (null)
  • Неопределен (undefined)
  • Объект (object)
Переменные

Давайте рассмотрим объявление переменной.

            var s; //объявление переменной

Здесь мы просто объявили переменную s.

Присвоение какого-то значения переменной

             var s = 1; //Присвоение значения переменной

Этот код читается так объявить переменную s и присвоить ей значение 1

Можно написать и так

             var s; //Объявили переменную
             var s = 1; //Присвоение значения переменной

Изменение значений переменных

            var w = "Hello"; //Присвоили значение
            w = "World"; //Изменили значение переменной

Замещение переменной (или копирование переменной)

            var a = 1; // Присвоили значение переменной a
            var b = 3; // Присвоили значение переменной b
            b = a; // Теперь b равно a (b = 1)

А теперь рассмотри типы переменных

            var s = 'String' ; // Строка
            var i = 1; // Число (number)
            var if = 1.025; // Число (number)
            var bool = true; // Логический (boolean)
            var boolf = false; // Логический (boolean)
            var nul = null; // Null
            var und;
            alert (und);// Undefined

Тут мы объявили переменные и присвоили им значени, про alert читай в взаимодействие

Важно Использование директивы var обязательно. Вслучае пропуска может произойти ошибка.
