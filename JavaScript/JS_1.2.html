Глава 1.2 - Конструкция if...else


Конструкция if

Конструкция if проверяет условие и если оно верно, то выполняет код

Синтаксис

            if (условие){
                //Код
            }

Пример

        if (5>2){
            alert ('Сработает');
        }
        if (5<2){
            alert ('Не сработает');
        }

Код читается так - Если 5 больше 2 тогда вывести alert ('Сработает');

Внимание Фигурные скобки рекомендуется использовать всегда т.к в случае пропуска может произойти ошибка

Преобразование к логическому типу

Конструкция if приводит условие к логическому типу

Число 0 , пустая строка , NaN, null, undefined - false; остальные значения - true.

        if (0){
            //не выполниться т.к 0 - false
        }
        if (1){
            //выполнится т.к 1 - true
        }
Конструкция if...else

Бывают случаи когда Вам надо выполнить код когда условие не верно, для существует конструкция if..else

Синтаксис

        if (условие){
            //код
        }
        else{
            //код
        }

Пример

        if (5<2){
            alert ('Не сработает');
        }
        else {
            alert ('Сработает');
        }

Код читается так - Если 5 меньше 2 вывести alert('Не сработает') иначе вывести alert ('Сработает')

Конструкция if...else if...else

Применяется когда надо проверить несколько условий

Синтаксис

        if (условие){
            //код
        }
        else if(условие 2){
            //код
        }
        else{
            //код
        }

Пример

        var a=2,b=3;
        if (a < b){
            alert ('a меньше b');
        }
        else if (a == b){
            alert ('a равно b');
        }
        else {
            alert ('a больше  b');
        }
Запись условий

Условия проверяемые конструкцией if можно записывать в переменные

        //Вариант 1
        if (2<3){
            //код
        }
        //Вариант 2
        var uslovie = 2<3;
        if (uslovie){
           //код
        }
Тернарный оператор

Тернарный оператор сокращает запись предыдущих конструкций

Получил свое название от того что принимет не один или два аргумента а целых три

Синтаксис

        условие ? значение1 : значение2

Пример

        var a = 4,b = 3;
        a < b ? alert ('a меньше b') : alert ('a больше или равно b') ;

При использвование конструкции if...else тогда код выглядил бы так

        var a = 4,b = 3;
        if (a < b){
             alert ('a меньше b');
        }
        else{
            alert ('a больше или равно b');
        }

Можно использовать несколько операторов ? 

        var a = prompt('Введите число от 1 до 3');
        a==1 ? alert ('1') : a==2 ? alert ('2') : a==3 ?  alert ('3') :  alert('незнаю');
