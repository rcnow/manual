Глава 1.2 - Конструкция if...else


Конструкция if

Конструкция if проверяет усовие и если оно верно, то выполняет код

Синтаксис

            if (условие){
                //Код
            }

Пример

        if (5>2){
            alert ('Сработает');
        }
        if (5<2){
            alert ('Не сработает');
        }

Код читается так - Если 5 больше 2 тогда вывести alert ('Сработает');

Бывают случаии когда Вам надо выполнить код когда условие не верно, для существует конструкция if..else

Синтаксис

        if (условие){
            //код
        }
        else{
            //код
        }

Пример

        if (5<2){
            alert ('Не сработает');
        }
        else {
            alert ('Сработает');
        }

Код читается так - Если 5 меньше 2 вывести alert('Не сработает') иначе вывести alert ('Сработает')
