Глава 1.3 - Конструкция switch


Конструкция switch

Сравнивает значение с различными вариантами и при совпадении выполняет соответствующий код, если совпадений не найдено выполняет код default

Синтаксис

            switch (значение){
                case 'вариант1':
                    //код
                    break;
                case 'вариант2':
                    //код
                    break;
                ....
                case 'вариантN':
                    //код
                    break;
                default:
                    //код
                    break;
            }

Значение сравнивается с вариантом и если они равны, то выполнение завершается для этого служит break , если совпадений не найдено выполняется default

Важно Сравнение строгое (===), значение и вариант точнее их типы должны быть строго равны

Пример

            var a = 2;
            switch (a){
                case 1:
                    alert('1');
                    break;
                case 2:
                    alert('2');
                    break;
                case 3:
                    alert('3');
                    break;
                default:
                    alert('неизвестно');
                    break;
            }
            // Попрубуйте изменить значение a на 1,2,3,4
        
            var a = 2;
            switch (a){
                case 1:
                    alert('1');
                    break;
                case 2:
                    alert('2');
                    break;
                case '3':
                    alert('3'); //Не выполниться т.к '3' - строка, а идет сравнение чисел
                    break;
                default:
                    alert('неизвестно');
                    break;
            }
            // Попрубуйте изменить значение a на 3

Важно break можно пропускать, тогда выполнение не завершиться и выполнение продолжиться при этом сравневнение игнорируется

            var a = 1;
            switch (a){
                case 1:
                    alert('1');
                case 2:
                    alert('2');
                case 3:
                    alert('3');
                default:
                    alert('неизвестно');
            }
            // Выведет все alert
        
            var a = 2;
            switch (a){
                case 1:
                    alert('1');
                case 2:
                    alert('2');
                case 3:
                    alert('3');
                default:
                    alert('неизвестно');
            }
            // Выведет все alert кроме alert('1');

Важно case можно объединять в группу

            var a = 2;
            switch (a){
                case 1:
                case 2:
                    alert('1 или 2');
                    break;
                case 3:
                    alert('3');
                    break;
                default:
                    alert('неизвестно');
                    break;
            }
