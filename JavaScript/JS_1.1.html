Глава 1.1 - Операторы


Операторы

Операторы делятся на:

  • Унарные - один операнд (отрицание, унарный минус)

  • Бинарные - два операнд (сложение, вычитание, умножение и т.д.)

  • Тернарные - три операнд ("условие ? выражение1 : выражение2")

Внимание Операнд — это то к чему применяется оператор.

Унарные

Унарный минус меняет знак

            var a = 1;
            a = - a; //Унарный минус
            alert (a);
            var b = 2;
            var c = 3;
            alert ( -(b+c) ); //Унарный минус, изменение знака после математической операции
            alert (-(-6));
Арифметические операторы

Арифметические операторы известны всем со школы

Важно Работа унарного оператора + и бинарного оператора + отличаются. Унарный плюс нечего не делает в отличии от бинарного плюса, но приводит значение к числовому типу (Number). См. код ниже

  • Плюс + (сложение)

  • Минус - (вычитание)

  • Умножение *

  • Деление /

  • Остаток от деления %

            var a = 6;
            var b = 3;
            alert (a+b); // Бинарный плюс
            alert (a-b); // Бинарный минус
            alert (a*b); // Бинарный оператор умножить
            alert (a/b); // Бинарный оператор деление
            alert (a%b); // Остаток от деления

Если бинарный оператор + применить к строке (string) то они будут объеденены

            var h = 'hello';
            var w = 'world';
            alert (h + w);

Важно Если любой из операндов строка (string) то второй операнд тоже приравнивается к строке. Такова особенность бинарного +. Остальные же операторы (-, *, /) приводят к числу

            var a = '4';
            var b = 2;
            alert (a+b); // Будет 42 а не 6
            alert (a-b); // 2
            alert (b*a); // 8
            alert (a/b); // 2

Работа унарного оператора +

            var a = '4';
            var b = '2';
            alert (a+b); // 42
            alert (+a + b); // 42 , b еще строка
            alert (+a + +b); // 6
Оператор присваивания

С оператором присваивания = мы познакомились еще в Главе 1

            var a = 4;
            var b = 2;
            var c = a + b + 6;
            alert (c); //12

Важно Возможно также присваивать по очереди

            var a,b,c;
            a = b = c = 4;
            alert (a);
            alert (b);
            alert (c);
Инкремент декремент

Инкремент ++ увеличение на 1

Декремент-- уменьшение на 1

            var a = 4;
            var b = 2;
            b++;
            alert (b); // 3
            a--;
            alert (a); // 3

Важно Операторы ++ и -- можно вызывать как перед переменной - постфиксная форма, так и после - префиксная форма

Внимание Делают они одно и тоже в независимости где стоит оператор т.е прибавляют 1

            var a = 3;
            a++; // префиксная форма
            ++a; // постфиксная форма
