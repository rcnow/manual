Глава 1.1 - Операторы


Операторы

Операторы делятся на:

  • Унарные - один операнд (отрицание, унарный минус)

  • Бинарные - два операнда (сложение, вычитание, умножение и т.д.)

  • Тернарные - три операнда ("условие ? выражение1 : выражение2")

Внимание Операнд — это то к чему применяется оператор.

Унарные

Унарный минус меняет знак

            var a = 1;
            a = - a; //Унарный минус
            alert (a);
            var b = 2;
            var c = 3;
            alert ( -(b+c) ); //Унарный минус, изменение знака после математической операции
            alert (-(-6));
Арифметические операторы

Арифметические операторы известны всем со школы

Важно Работа унарного оператора + и бинарного оператора + отличаются. Унарный плюс нечего не делает в отличии от бинарного плюса, но приводит значение к числовому типу (Number). См. код ниже

  • Плюс + (сложение)

  • Минус - (вычитание)

  • Умножение *

  • Деление /

  • Остаток от деления %

            var a = 6;
            var b = 3;
            alert (a+b); // Бинарный плюс
            alert (a-b); // Бинарный минус
            alert (a*b); // Бинарный оператор умножить
            alert (a/b); // Бинарный оператор деление
            alert (a%b); // Остаток от деления

Если бинарный оператор + применить к строке (string) то они будут объеденены

            var h = 'hello';
            var w = 'world';
            alert (h + w);

Важно Если любой из операндов строка (string) то второй операнд тоже приравнивается к строке. Такова особенность бинарного +. Остальные же операторы (-, *, /) приводят к числу

            var a = '4';
            var b = 2;
            alert (a+b); // Будет 42 а не 6
            alert (a-b); // 2
            alert (b*a); // 8
            alert (a/b); // 2

Работа унарного оператора +

            var a = '4';
            var b = '2';
            alert (a+b); // 42
            alert (+a + b); // 42 , b еще строка
            alert (+a + +b); // 6
Оператор присваивания

С оператором присваивания = мы познакомились еще в Главе 1

            var a = 4;

            var c = a + b + 6;
            alert (c); //12

            var a,b,c;

Инкремент декремент

Инкремент ++ увеличение на 1

Декремент-- уменьшение на 1

            var a = 4;
            var b = 2;
            b++;
            alert (b); // 3
            a--;
            alert (a); // 3

Важно Операторы ++ и -- можно вызывать как перед переменной - постфиксная форма, так и после - префиксная форма

Внимание Делают они одно и тоже в независимости где стоит оператор т.е прибавляют 1

            var a = 3;
            a++; // префиксная форма
            ++a; // постфиксная форма

Между этими видами записи существует разница. Разница быдет заметна только когда нам нужно не только увеличить или уменьшить переменную, но и использовать результат в этом же выражении

            var a = 3;
            alert ('a = '+ a++); // 3
            //теперь a = 4
            alert ('Теперь a = ' + a)
            var b = 3;
            alert ('b =' + ++b) // 4

Оператор ++ -- можно пременять и в выражениях

            var a = 3;
            var result = 3 + ++a;
            alert (result); // 7
Побитовые операторы

  • Побитовое И (AND)

  • Побитовое ИЛИ (OR)

  • Побитовое исключающее ИЛИ (XOR)

  • Побитовое НЕ (NOT)

  • Левый сдвиг

  • Правый сдвиг, переносящий знак

  • Правый сдвиг с заполнением нулями

Записываются они так

            	a & b // AND
             	a | b // OR
            	a ^ b // NOR
            	~ a // NOT
            	a << b // Левый сдвиг
            	a >> b // Правый сдвиг, переносящий знак
            	a >>> b // Правый сдвиг с заполнением нулями

Логические значения

Существует 2 вида логических значения это true и false

true - означает истина

false - означает ложь

Операторы сравнения

  • Больше и меньше

  • Больше, меньше или равно

  • Равно

  • Не равно

Пишутся они так

            a > b; // Больше
            a < b; // Меньше
            a <= b; // Меньше или равно
            a >= b; // Больше или равно
            a == a; // Равно
            a != b; // Не равно

Сравнение чисел

            var a = true; //тип boolean (логическое значание true);
            var b = 5>4;
            var c = 5<4;
            alert (b); //true
            alert (c);//false
            alert (a == c); //false т.к true не может быть равно false. alert(true == false)
            alert (a == b); //true
            alert (a != c);  //true

Сравнение строк

            alert ('б' > 'а'); //true

Внимание Буквы сравниваются в алфавитном порядке. Какая буква в алфавите позже — та и больше.

При сравниение строк сначала сравнивается первая буква если она больше то дальше сравнение не проиходит, если одинаковы то сравнение идет дальше.Если же в одной сторке отсутсвует буква, а в другой она есть то больше > та строка в которой буква есть

            alert ('Банан' > 'Ананас');// true
            alert ('Вася' > 'Ваня'); //true
            alert ('Дом' > 'Домик'); //false

Важно При сравнении значения преобразуются к числам. Исключение: когда оба значения — строки, тогда не преобразуются.

Внимание При сравнивание логических значений с числом они(логические значения) преобразуются к числам true - 1, false -0

Для приведения значения к числу ипользуйте унарный плюс. Есть и другие способы преобразования о них читайте тут глава x

            alert ('4' > 3); //true
            alert ('0' > 700); //false
            alert ('2' > '14');//true
            alert ('21' <' 1000');//false
            alert (true == 0); //false
            alert (+'21' < +'1000');//true

Важно Для проверки равенства без преобразования типов используются операторы строгого равенства === и !== .

             alert (true == 1); //true
             alert (true === 1); //false
Логические операторы

Логические операторы деляться:

  • И - &&

  • ИЛИ - ||

  • НЕ - !

Важно И && возвращает true, если оба аргумента истинны, иначе — false

            alert( true  && true ); // true
            alert( false && true ); // false
            alert( true  && false); // false
            alert( false && false); // false

Важно ИЛИ || возвращает true если хотя бы один из аргументов true, иначе — false».

            alert( true  || true ); // true
            alert( false || true ); // true
            alert( true  || false); // true
            alert( false || false); // false

Важно НЕ ! Выполняет логическое отрицание выражения

            alert( !true); // false
            alert( !false ); // true

Важно Двойное НЕ !! может использоваться для преобразования в тип boolean

            var str = "hello";
            alert (str); // hello
            alert (!!str); // true
