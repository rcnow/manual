Глава 1.4 - Циклы for, while и другие


Циклы предназначены для повторения участка кода. Повторение цикла называют еще итерация

Цикл for

Цикл for является самым распространненым.

Синтаксис

            for (переменная ; условие; действие){
                //код цикла
            }

переменная - начало цикла

условие - определяет условие цикла

действие - выполняется каждый раз когда код цикла выполнен. Т.е пока не выполниться условие.

Важно Точка с запятой (;) обязательны

Пример

            var i;
            for (i=1;i<=3;i++){
                alert (i);
            }

Это пример выведет значения от 1 до 3. В этом примере 3 итерации

Важно Работает цикл так переменная (i=1) выполняется при входе в цикл, условие (i<=3) проверяеться перед каждым повторением цикла(итерацией), действие(i++) выполняется после каждой итерации, но перед проверкой условия

Внимание Переменную можно объявить и внутри цикла

            for (var i=1;i<=3;i++){
                alert (i);
            }

Важно Любая часть цикла может быть пропущенна

            var i=1;
            for (;i<=3;i++){
                alert (i);
            }//убрали переменную из цикла

            var i=1;
            for (;i<=3;){
                alert (i);
                i++;
            }//убрали действие из цикла

            var i=1;
            for (;;){
                alert (i);
            }//убрали все из цикла

Цикл while

Синтаксис

            while(условие){
                //код цикла
            }

Код цикла будет выполняться до тех пор пока условие не будет верно

Пример

            var i=1;
            while(i<=3){
                alert (i);
                i++;
            }

Это пример выведет значения от 1 до 3

Аналогичный код можно написать используя цикл for

            var i=1;
            for (;i<=3;){
                alert (i);
                i++;
            }
Цикл do...while

Синтаксис

            do{
                //код выполнения
            }
            while (условие);

Отличается от цикла while тем что сначала выполняется код а затем проверяется условие

Не самый распространенный вариант

            var i=1;
            do{
                alert(i);
                i++;
            }
            while(i<=3);

Внимание Так условие приводиться к логическому типу уместно писать while (i) вместо while (i!=0)

            var i=3;
            while(i){
                alert (i);
                i++;
            }
Бесконечный цикл

Иногда нам нужно чтобы цикл не заканчивал свою работу при выполнении условия для этого можно использовть циклы for или while

            for(;;){
                ////код который будет выполнен
            }
        
            while(true){
                //код который будет выполнен
            }
