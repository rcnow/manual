Взаимодейстрие с пользователем

Для взаимодействия с пользователем глобальные методы. Что такое методы глава x ?

  • alert

  • prompt

  • confirm

Рассмотрим каждый в отдельности. Самый простой это alert, который выводит сообщение и приостанавливает работу пока пользователь не нажмет кнопку OK

Синтаксис

            alert ('Ваше сообщение');

Пример

            alert ('Hello World');
            var a = 3;
            alert (a);

Метод prompt выводит сообщение с заголовком title и полем ввода со значение value

Синтаксис

            var result = prompt ('title',value);

Пример

            var result = prompt ('Сколько Вам лет?',55);
            alert (result);

Внимание значение value может отсутсвовать

Пример

            var result = prompt ('Сколько Вам лет?');
            alert (result);

Важно При использование браузера Internet Explorer в поле ввода выведет undefined. Так что лучше использовать следующий код 

            var result = prompt ('Сколько Вам лет?','');
            alert (result);

Внимание Если пользователь нечего не введет результатом будет пустая строка, если нажмет CANCEL - null

Важно prompt возвращает строку (string)

И последний метод confirm выводит окно с вопросом и двумя кнопками OK и CANCEL. При нажатие OK будет true, а при нажатие CANCEL - false

Синтаксис

            var result = confirm (вопрос);

Пример

            var result = confirm ('Вам есть 18 лет?');
            alert (result);
